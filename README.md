# 大同大學科學創意暨開源服務社

這是一群努力為現實世界做tldr的人們聚集的地方，無論你是對生活科學、電腦科學或是單純對服務有興趣，這個社團都絕對適合你！
不論你是哪個學院，這裡都會是一個適合你發展的地方！我們會從零開始，手把手帶你進入科學開源的新世界
在這個社團中，因為開源的關係，我們會創造很多的機會，給各位也夢想和理想的青年！
